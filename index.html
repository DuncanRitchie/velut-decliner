Duncan’s Latin declensions for velut





About

velut is my Latin vocabulary website. I review all words before adding them to velut. This is made faster by generating forms using this webpage.

If you’re not me (Duncan Ritchie), you’re unlikely to have much use for this page. If you’re interested in Latin vocabulary though, you should definitely take a look at velut!

Note on the word “lemma”

The lemma of a word is the form given as the headword in dictionaries; for nouns it’s usually the nominative singular.

How to use

Enter the lemmata of Latin nouns/adjectives/adverbs into the first input box. These can be separated by spaces, line-breaks, commas, or semicolons. I provide some sample words that you can load if you want.

For words of the third declension, please enter both the nominative singular and the genitive singular, separated by a slash, eg “amor/amōris”. (But if the nominative singular is the same as the genitive singular, with or without the -is ending, it’s fine to just give the nominative singular, eg “avis” and “cōnsul” will yield the same output as “avis/avis” and “cōnsul/cōnsulis”.)

This trick (of supplying the genitive singular after a slash) also works for fourth declension nouns with nominative singular -us, so you can type either “manus” or “manus/manūs”.

Then, click the “Start choosing declensions” button. A table will appear listing every word you put in, alongside a declension — such as “3rd, noun, consonant stem” — that has been automatically determined for each word.

This page determines the declension based on the final letters of the word, so is frequently wrong. You can, however, edit the declension, by deleting the incorrect text and choosing a different declension from the list that appears.

Once you are happy with the table of lemmata and declensions, click “Decline”. A list of all the forms will appear in the box underneath. Forms that are stressed on the ending may appear if the “Include stressed endings” tickbox is ticked. The lemma that each form came from may appear beside each lemma (separated by a tab), depending on whether the “Output lemma” tickbox is ticked.

Finally, you can click the “Copy to clipboard” button.

Brackets in input

In the input, you can include one of the following abbreviations in brackets after any word: “adj”, “adv”, “n”, or “prn”. These stand for “adjective”, “adverb”, “(non-proper) noun”, and “proper noun”, respectively.

This page takes the abbreviation into account when automatically assigning a declension to the word, though they are optional.

Marking a word as a proper noun has the same effect on which declension is automatically assigned as marking it as a non-proper noun. “Deus[prn]” and “deus[n]” will both be assigned “2nd, noun, masculine/feminine, -us”, for instance.

If square brackets are used, the abbreviation will appear in the final output with the lemma, if the “Output lemmata?” tickbox is ticked.

If round brackets are used, the abbreviation will not appear in the output.

Caveats

Please note: you need to look over all the declined forms that have been generated, because this webpage is quite rudimentary and not every inflection makes sense for every lemma.

Also, this page assumes you’re capable of entering letters with macra (the diacritics on ĀĒĪŌŪȲāēīōūȳ) into the first input box.

If the lemma (or genitive singular) does not end with the letters this page expects, all the forms will probably be wrong.

For the third declension, and for fourth declension nouns with lemmata in -us, you can input the genitive singular after a slash and the output may be what you want. For fourth declension nouns with nominative singular -ū, and for the other declensions (1st/2nd/5th), please only supply the nominative singular. That is, “amor/amōris” and “manus/manūs” will yield the correct output (even if you don’t manually change the declension in the table), but “terra/terrae”, “corvus/corvī”, “cornū/cornūs” will not.

Some Latin nouns are more irregular in their endings, so this page won’t handle them correctly. This is true for 3rd-declension nouns with -im or -īs accusative forms (eg “febris”) and some Greek proper nouns (eg “Ēchō”), among others.

You should also be aware that this page generates singular and plural forms for all lemmata, but some nouns don’t exist in the singular or don’t make sense in the plural. For each adjective or adverb, comparative or superlative forms may be generated, but these aren’t always attested.

What I use this for

This webpage is convenient for me because much of the data for velut (my Latin dictionary) is generated by and stored in a large Excel file. For each word, I need to generate several values based off the word and the lemma (or lemmata) that the word belongs to. A sheet in my Excel file generates the values I need, based off whatever text is pasted into two of the columns (the “word” and “lemmata” columns).

Therefore, I can paste the output from this webpage into my Excel file, check everything is correct (it probably isn’t), and that’s how I generate everything velut needs for each word.

The de-Excellation of velut

The Excel file that stores, generates, and manages all the data for velut is very large and complex. This is a problem for several reasons — loading and saving the file takes a long time, occasionally it crashes and I lose work, I can’t choose to make some data or some functionality publicly visibly without making it all publicly visible…

That’s why I am in the process of reducing velut’s reliance on Excel, by re-creating all the functionality in the velut Excel file as webpages or websites and by exporting all the data to MongoDB databases or Json files to be used by the webpages/sites.

This webpage is a part of that, since it replaces an Excel sheet that was generating declined forms.

I have a similar Excel sheet that generates conjugated forms of verbs. I’ve therefore created a webpage similar to this one, but for conjugating verbs instead of declining nouns and adjectives.

As more and more of velut gets de-Excellated, further changes will be needed to this page. For now, it’s okay.
